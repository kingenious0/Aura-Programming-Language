# Aura Language Specification v1.0

## 1. Variables (Data)
Syntax: "The [object]'s [property] is [value]"
Example: "The user's name is 'Akwasi'"

## 2. Actions (Functions)
Syntax: "When [event], [action]"
Example: "When clicked, display 'Success'"

## 3. UI (Elements)
Syntax: "Create a [element] with the text [string]"
Example: "Create a button with the text 'Submit'"

## 4. Theme (Global Style)
Syntax: "Use the [theme name] theme"
Example: "Use the dark theme"
