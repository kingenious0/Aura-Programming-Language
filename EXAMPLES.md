## Tips for Writing Aura Code

1. **Start with a theme** - Always begin with `Use the [theme] theme`
2. **Structure your content** - Use headings and paragraphs to organize
3. **Add interactivity** - Use `When clicked` to make buttons interactive
4. **Use comments** - Lines starting with `#` are ignored
5. **Keep it simple** - Aura is designed to be readable and straightforward

## Common Patterns

### Hero Section

Create a heading with the text 'Your Main Message'

## Running These Examples

Save any example to a `.aura` file and run:

```bash
python transpiler/transpiler.py your_file.aura
```

Then open the generated `index.html` in your browser!
